Machine Failure Risk Prediction App
This project presents a machine learning model designed to predict machine failure based on process variables such as temperature, torque, rotational speed, and tool wear.

An interactive Streamlit application allows users to input operating conditions and receive real-time risk predictions.

Dataset
AI4I 2020 Predictive Maintenance Dataset (Kaggle)

This dataset contains simulated industrial process data, including temperature, torque, rotational speed, and failure indicators.

Key Features
Machine learning model with class imbalance handling
Identification of key failure drivers
Definition of safe operating ranges
Interactive prediction tool using Streamlit
Objective
The goal is to support early detection of high-risk conditions and improve operational decision-making in industrial environments.
